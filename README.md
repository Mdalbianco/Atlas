# Atlas

Atlas è una piattaforma di trading automatico per criptovalute.

## Obiettivo

Costruire un sistema modulare che sia in grado di:

- analizzare il mercato
- individuare opportunità
- gestire il rischio
- operare tramite Kraken
- essere controllato da Telegram
